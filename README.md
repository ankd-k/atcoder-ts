# atcoder-ts
atcoder TypeScript template

## HowToUse
1. clone this repository
2. run `yarn i`
3. edit main function at src/main.ts
4. run `yarn test` to test your program
5. copy and paste to atcoder's editor

## Feature
- testing program on your local pc
- enable using stdin
